# Original Game

## ゲーム制作方針
**ゲームの種類:** シューティングゲーム 

**内容:** 制限時間内に相手の陣地にダメージを与えて破壊することを競う

## ゲームの勝利条件
**1.** 相手の基地のHPを0にする

**2.** 敵プレイヤーを先に5回撃破する

**3.** 制限時間後に1.・2.の条件を満たしていない場合、お互いの基地の残りHPを比較して多い方
